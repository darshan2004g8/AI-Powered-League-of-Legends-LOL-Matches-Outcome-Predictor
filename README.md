<h2>🎮 AI League of Legends Match Outcome Predictor</h2>

An AI-powered League of Legends Match Outcome Predictor that uses Machine Learning and Deep Learning (PyTorch) to predict whether a team will Win or Lose based on pre-game and in-game match statistics.
This project demonstrates the complete machine learning pipeline—from data preprocessing and feature engineering to model training, evaluation, and prediction—using Logistic Regression implemented in PyTorch.

<h2>🚀 Project Overview</h2>

The objective of this project is to build an intelligent prediction system capable of estimating the outcome of a League of Legends match using historical match data.

The workflow includes:

1. Loads League of Legends match dataset
2. Cleans and preprocesses data
3. Feature scaling using StandardScaler
4. Logistic Regression built with PyTorch
5. Binary classification (Win / Loss)
6. Training loss visualization
7. Accuracy evaluation
8. Predicts outcomes for unseen matches
